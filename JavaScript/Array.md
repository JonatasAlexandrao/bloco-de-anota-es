
- Declarando um array
    `const a = []`

- Declarando um array
    `const array = [1, 2, 3]`

- Destructuring
  `const [x,, y] = array`
  Ele vai criar uma const pra x que vai receber o primeiro valor do array anterior, a ',,' indica pra ele pular aquela posição e o y vai receber o terceiro lugar do array.
  `x <- 1`
  `y <- 3`
