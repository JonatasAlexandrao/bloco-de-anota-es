
# Convertendo String para Numero:

  ## String para Decimal
  `parseInt(num1)`

  ## String para Float
  `parseFloat(num1)`

  ## Numero para String
  `1 + ""` = "1"

# Arredondar casas decimais:
  `num = 20.339999999`
  `num.toFixed(2)`
  = 20.34

# Trocando caracteres por outros:

  # replace()
  Troca o primeiro parametro pelo segundo.

    ```
    const n = "20,22";
    n.replace(",",".");

    resultado = "20.20"
    ```
